# PilotWave

Pilot wave mechanics using python. The main entry point is `propagation.py`:

    $ python propagation.py

Parameters for the code can be found and altered in `constants.py`

# Requirements

This code is written in `python 3.7.1`

1. `scipy==1.2.1`
2. `numpy==1.16.2`
3. `matplotlib==2.2.3`
